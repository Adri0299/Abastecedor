Acerca de


El abastecedor requiere de un sistema de Inventario el cual le permite tener un mayor control de los productos que se encuentran en el abastecedor; esto ayudara a que los empleados o jefe conozcan la cantidad de productos, marca y precio para brindar un servicio mas veloz.


Ademas se tendra un mayor control de los provedores, contacto y fecha de visita, para conocer cuando se debe de pedir los productos y tener un mejor manejo del tiempo.
